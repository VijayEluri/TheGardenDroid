Welcome to the Garden Droid API

API:
  • Save Sensor Data - POST
        ./rest/gardenDroidData/saveData Used by monitoring systems to report data.
        POST body example: 
    {
"dataValue": 0,
"sensorType": "TEMPERATURE",
"tempF": 58.65,
"tempC": 14.80
}
  • Current Conditions - GET
    ./rest/sensor/current Gets latest readings for each sensor type.
  • Temp History - GET
    /rest/sensor/temp-history Gets previous temp history.
